Who I Am

I'm a person who loves to learn and be creative, but since I can't draw to save my life, I dove into the world of software development to use my creativity to solve logic puzzles. Playing a part in creating the future of the tech world lights me up, and I never sleep better than after a day of solving the problems of our time. When I'm not grinding the keyboard, I'm passionate about traveling to experience the world, supporting my Muay Thai gym & team, enjoying live music, and tending to my wildflower garden.

What I've Done

I possess an Associate's Degree in Computer Science from Brown College, where I studied everything from software engineering to corporate network development, gaining hands-on experience with all forms of IT professional needs. I also have my Full Stack Software Engineering Certification from Prime Digital Academy, a deep-immersion full-time program where I built full CRUD web applications with the latest languages while being entirely remote.

Currently I am a freelance team lead for a team of three developers, doing contractual work for our clients via ad hoc requests for flat-rate fees. In addition to that, I am also a Code Reviewer at Prime Digital Academy, where I analyze student’s code and provide technical feedback and direction via empathetic code reviews.

Add User Registration Wizard

A project done as a Front-end Build Challenge, with the goal of building a 'User Registration Wizard' as close as possible to the 3 mock-up images provided, following a set of requests for each page. The wizard has the user enter information, select a User Group, and then verify the data before submission.

Client Work: PrimX Order Portal

PrimX NA is a national supplier of a unique concrete product. However, PrimX clients were still placing orders with spreadsheets. We performed an order platform overhaul, allowing clients to create and submit estimates, while giving PrimX admin a view of their business pipeline on the back-end.

The Movies Saga

The Movies Saga will allow a user to create a list of movies they like, by entering a name, description, selecting a category, and uploading an image. The users will then be able to edit these details by clicking on a movie in the list and using the edit button to save and update, with total data permanence.

Snapshot Tiles

Snapshot Tiles is an app similar to Instagram or Facebook, where a user can upload their own pictures with descriptions, and viewers can click to "Like" the photo. When a photo is clicked, it flips to show the text description with conditioning rendering, and can be flipped back with another click.
