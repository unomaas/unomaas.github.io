Who I Am

I'm a person who loves to learn and be creative, but since I can't draw to save my life, I dove into the world of software development to use my creativity to solve logic puzzles. Playing a part in creating the future of the tech world lights me up, and I never sleep better than after a day of solving the problems of our time. When I'm not grinding the keyboard, I'm passionate about traveling to experience the world, supporting my Muay Thai gym & team, enjoying live music, and tending to my wildflower garden.

What I've Done

I possess an Associate's Degree in Computer Science from Brown College, where I studied everything from software engineering to corporate network development, gaining hands-on experience with all forms of IT professional needs. I also have my Full Stack Software Engineering Certification from Prime Digital Academy, a deep-immersion full-time program where I built full CRUD web applications with the latest languages while being entirely remote.

In previous roles, I've served as an Executive Assistant for the largest energy-efficiency rating company in Minnesota, where I worked with all executive staff to help manage the company. There I collaborated with the in-house software development team for quality assurance testing of all new features, providing detailed end-user feedback and troubleshooting.

Prior to that, I was an Assistant Manager for a kickboxing gym, where I coached class sizes of up to 30 students daily while maintaining a fun atmosphere. It was there that I gained my love of teaching and discovered my competitive side, while promoting physical wellbeing and self-defense knowledge.

What I've Created

Please feel free to look through some of my more recent projects. Everything shown below are full-stack web apps using CRUD methodology to have the front-end interact with a virtual server, built entirely by myself.

Click on the thumbnails to see a quick demo GIF of each project. Feedback is always welcome!

Pack Me Up

Made for all the people like me who travel often and pack heavy, Pack Me Up aims to be the world's best packing app, and will be taking a suitcase of stress off of your mind by letting you organize everything you'd ever need to travel with into neat little boxes. Never leave anything behind again!

  • My Presentation

The Movies Saga

The Movies Saga will allow a user to create a list of movies they like, by entering a name, description, selecting a category, and uploading an image. The users will then be able to edit these details by clicking on a movie in the list and using the edit button to save and update.

  • The Code

Feedback App

Designed to mimic the feedback system of Prime Digital Academy, the app functionality will take the user through a simple "feedback" process, asking them to leave feedback on how they're feeling, how they're understanding content, how well supported they feel, and if there's any comments.

  • The Code

Snapshot Tiles

Snapshot Tiles is an app similar to Instagram or Facebook, where a user can upload their own pictures with descriptions, and viewers can click to "Like" the photo. When a photo is clicked, it flips to show the text description with conditioning rendering, and can be flipped back with another click.

  • The Code

To Do List App

This is a web based application that interacts with a virtual server and database to run a simple "To Do List" tracker. The goal of this project was to get more hands on experience building an app with full CRUD functionality, using jQuery, Bootstrap, AJAX, and HTML/CSS/JS.
